Node.js and Express
Installation Tutorial

As an asynchronous event-driven JavaScript runtime, Node.js is designed to build scalable network applications. In the following "hello world" example, many connections can be handled concurrently. Upon each connection, the callback is fired, but if there is no work to be done, Node.js will sleep.

From official Node.js website

Download Node.js

1) Go to nodejs.org to download the source code accordingly to the operating system.

Install Node.js

2) Run the Node.js Setup installer.

3) Choose the features you would like to install on your computer. Follow the prompts of the installer.

Testing

1) Test Node: To see if Node is installed on your computer, open command prompt and run the command node -v in the terminal. This will show the current installed version of Node on your computer.

2) Test NPM: To see if NPM is installed on your computer, type and run the command npm -v in the terminal. This will show the the version of NPM installed.
