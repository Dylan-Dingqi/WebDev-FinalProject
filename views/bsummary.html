Node.js
Back-End Summary

As an asynchronous event-driven JavaScript runtime, Node.js is designed to build scalable network applications. Upon each connection, the callback is fired, but if there is no work to be done, Node.js will sleep.

Node.js Popularity

Over the years, Node.js's popularity has steadily risen. This is due to it's performance and benefit of allowing programmers to build end-to-end Javascript applications.
