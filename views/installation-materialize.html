Materialize
Installation Tutorial

Created and designed by Google, Material Design is a design language that combines the classic principles of successful design along with innovation and technology. Google's goal is to develop a system of design that allows for a unified user experience across all their products on any platform.

From official Materialize website

Download Materialize

Go to https://materializecss.com/getting-started.html to download the source code.

Link to Materialize from HTML

Put the source code folder in the project directory, in public folder, it is where we are hosting all static files.
In our server index.js We used the following line of code to host them:

app.use('/public', express.static('public'));

And now in any HTML files, we can use a link tag to link to Materialize CSS library with path /public/materialize/.
